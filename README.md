# 🌿 কৃষি মিত্র - সম্পূর্ণ ইনস্টলেশন গাইড

## ফাইল কাঠামো
```
krishi-mitra/
├── index.html       ← মূল ফ্রন্টেন্ড (সব পেজ)
├── api.php          ← সম্পূর্ণ ব্যাকএন্ড API
├── config.php       ← ডেটাবেস কনফিগারেশন
├── database.sql     ← ডেটাবেস স্কিমা ও সিড ডেটা
├── uploads/         ← পণ্যের ছবি (স্বয়ংক্রিয়ভাবে তৈরি হবে)
└── README.md
```

## ইনস্টলেশন ধাপ

### ১. সার্ভার প্রয়োজনীয়তা
- PHP 7.4+ (PDO MySQL সহ)
- MySQL 5.7+ বা MariaDB
- Apache/Nginx (XAMPP/WAMP/Laragon)

### ২. ডেটাবেস তৈরি
```sql
-- phpMyAdmin বা MySQL CLI তে চালান:
source database.sql
```

### ৩. কনফিগারেশন
`config.php` ফাইলে পরিবর্তন করুন:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'আপনার_ডেটাবেস_ইউজার');
define('DB_PASS', 'আপনার_পাসওয়ার্ড');
define('DB_NAME', 'krishi_mitra');
define('BASE_URL', 'http://localhost/krishi-mitra/');
```

### ৪. সার্ভারে আপলোড
সব ফাইল `htdocs/krishi-mitra/` বা `www/krishi-mitra/` ফোল্ডারে রাখুন।

### ৫. পারমিশন
```bash
chmod 777 uploads/
```
